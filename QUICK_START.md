# دليل سريع - Money with Basel 💰

## 🚀 البدء الفوري (Getting Started)

### الخطوة 1: فتح التطبيق
```
1. افتح ملف `index.html` في متصفح ويب
2. اختر تسجيل دخول أو جرب النسخة التجريبية
```

### الخطوة 2: تثبيت التطبيق (اختياري)
```
1. انقر على الخيارات (⋮) في المتصفح
2. اختر "تثبيت التطبيق" أو "Install"
3. سيظهر أيقونة على شاشتك الرئيسية
```

## 📱 الميزات الأساسية

### 1️⃣ الشاشة الرئيسية (Dashboard)
عند فتح التطبيق ستجد:

| العنصر | الوصف |
|--------|--------|
| **صافي الثروة** | مجموع أرصدة جميع حساباتك |
| **الإيرادات** | مجموع دخلك |
| **المصاريف** | مجموع نفقاتك |
| **حساباتك** | قائمة محافظك وأرصدتها |
| **آخر العمليات** | آخر 5 حركات مالية |
| **ملخص الميزانية** | توزيع المصاريف حسب الفئات |

### 2️⃣ إضافة عملية مالية

**طريقة سريعة:**
```
1. انقر على الزر ➕ الموجود أسفل الشاشة
2. اختر نوع العملية (مصروف أو إيراد)
3. اختر الفئة المناسبة (مقاضي، فواتير، مطاعم...)
4. أدخل المبلغ
5. اختر الحساب (محفظتي، حساب البنك...)
6. حدد التاريخ
7. أضف ملاحظة (اختياري)
8. انقر "إضافة العملية"
```

**مثال:**
```
النوع: مصروف
الفئة: 🛒 مقاضي
المبلغ: 45.50 د.ا
الحساب: محفظتي
التاريخ: 2024-01-15
الملاحظة: شراء من السوبر ماركت
```

### 3️⃣ إنشاء التزام ثابت (Recurring)

**عند إضافة عملية، إذا كانت متكررة:**
```
1. فعّل خيار "التزام ثابت"
2. حدد تاريخ الانتهاء
3. اختر نمط التكرار:
   - يومي (كل يوم)
   - أسبوعي (كل أسبوع)
   - شهري (كل شهر) ⭐ الأشهر استخداماً
   - سنوي (كل سنة)
```

**أمثلة:**
```
مثال 1 - قسط سيارة:
- المبلغ: 200 د.ا
- التاريخ: 15 من كل شهر
- التكرار: شهري
- الانتهاء: 2025-01-15 (بعد سنة)

مثال 2 - جمعية:
- المبلغ: 100 د.ا
- التاريخ: 1 من كل شهر
- التكرار: شهري
- الانتهاء: 2024-12-01 (10 أشهر)

مثال 3 - راتب شهري:
- المبلغ: 1500 د.ا
- التاريخ: 1 من كل شهر
- التكرار: شهري
- الانتهاء: مفتوح
```

## 🏦 إدارة الحسابات

### إنشاء حساب جديد
```
1. انتقل إلى قسم "الحسابات" من الشريط السفلي
2. انقر على "➕ حساب جديد"
3. ملأ البيانات:
   - اسم الحساب (مثلاً: محفظتي)
   - نوع الحساب:
     * 💵 نقد (محفظة)
     * 🏦 حساب بنكي
     * 💳 بطاقة ائتمان
     * 🏪 حساب توفير
   - الرصيد الأولي
   - العملة (د.ا، دولار، يورو)
4. انقر "إنشاء الحساب"
```

### أنواع الحسابات الموصى بها
```
للفرد:
├── 💵 محفظة نقد
├── 🏦 حساب البنك الرئيسي
└── 🏪 حساب التوفير (اختياري)

للعائلة:
├── 💵 محفظة مشتركة
├── 🏦 حساب عائلي
├── 💳 بطاقة المشتريات
└── 🏪 صندوق الطوارئ
```

## 📊 الفئات المرنة

### الفئات الافتراضية (Presets)

**المصاريف:**
```
🛒 مقاضي    - تسوق، بضائع
⚡ فواتير   - كهرباء، ماء، إنترنت
🍔 مطاعم    - وجبات، مقاهي
🚗 مواصلات  - سيارة، تاكسي، مواصلات
🎬 ترفيه    - أفلام، ألعاب، رياضة
⚕️ صحة      - دكتور، أدوية
```

**الإيرادات:**
```
💼 راتب     - الراتب الشهري
📈 استثمارات - عوائد، أرباح
```

### إضافة فئة مخصصة
```
1. اذهب إلى "الإعدادات"
2. ابحث عن قسم "فئات العمليات"
3. انقر "➕ إضافة فئة"
4. اختر:
   - الاسم (مثلاً: جمعية)
   - الإيموجي (👨‍👩‍👧‍👦)
   - النوع (مصروف/إيراد)
   - اللون (للتمييز البصري)
```

## ⚙️ إعدادات الراتب

### تعريف الراتب الشهري
```
1. اذهب إلى "الإعدادات"
2. ابحث عن "الراتب الشهري"
3. أدخل:
   - قيمة الراتب (مثلاً: 1500 د.ا)
   - يوم الدفع (من الشهر: 1-31)
4. انقر "حفظ الإعدادات"
```

**سيقوم النظام بـ:**
```
✅ إضافة مبلغ الراتب تلقائياً في اليوم المحدد
✅ تحديث رصيد الحساب المربوط
✅ إظهار الراتب في سجل العمليات
```

## 📱 الاستخدام على الهاتف

### الملاحة
```
الشاشة الرئيسية      → بيت
  ↓
العمليات            → قائمة
  ↓
الحسابات            → محفظة
  ↓
الإعدادات           → ترس

استخدم الشريط السفلي للتنقل بين الأقسام
```

### الزر العائم (FAB)
```
الزر الوردي الكبير أسفل اليمين:
- اضغط عليه لإضافة عملية جديدة
- يظهر في:
  - الشاشة الرئيسية
  - قسم العمليات
  - قسم الحسابات
```

## 🆘 استكشاف الأخطاء

### المشكلة: بيانات لا تظهر
```
الحل:
1. أعد تحميل الصفحة (F5)
2. تحقق من أن localStorage مفعل
3. احذف ذاكرة التخزين المؤقتة وحاول مجدداً
```

### المشكلة: لا تستطيع إضافة عملية
```
الحل:
1. تأكد من إنشاء حساب أولاً
2. تحقق من أن جميع الحقول مملوءة
3. تأكد من المبلغ صحيح (أكبر من 0)
```

### المشكلة: التطبيق بطيء
```
الحل:
1. أغلق البرامج الأخرى
2. امسح ذاكرة التخزين المؤقتة
3. أعد تحميل الصفحة
```

## 💡 نصائح احترافية

### النصيحة 1: استخدام الملاحظات
```
أضف ملاحظات واضحة:
❌ "35.50"
✅ "شراء حاجيات من الكارفور"

❌ "100"
✅ "قسط سيارة - شهر 1"
```

### النصيحة 2: تنظيم الفئات
```
ابدأ بالفئات الأساسية ثم أضف حسب احتياجك
لا تكثر من الفئات (5-10 كافية)
```

### النصيحة 3: التزامات الشهرية
```
عريّف كل التزاماتك الثابتة:
- الإيجار
- الفواتير
- الجمعيات
- الاشتراكات
```

### النصيحة 4: الفحص الدوري
```
كل نهاية شهر:
✅ راجع الميزانية
✅ فعّل التزامات جديدة
✅ احذف التزامات انتهت
✅ ضبط الأهداف
```

### النصيحة 5: الأرصدة
```
تطابق أرصدة التطبيق مع أرصدة البنك
إذا اختلف:
1. فتّش عن عمليات ناقصة
2. تحقق من التحويلات بين الحسابات
3. راجع الوقت (قد يكون فرق زمني)
```
