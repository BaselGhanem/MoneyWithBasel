# Money with Basel 💰

**لوحة قيادة مالية احترافية للمستخدم العربي**

> تطبيق ويب تقدمي (PWA) يجمع بين احترافية Monarch Money وبساطة الاستخدام مع الروح المحلية الأردنية

---

## 🎯 الرؤية والهدف

Money with Basel ليس مجرد جدول مصاريف، بل هو **"لوحة قيادة لثروتك"** مصممة لتعطيك:
- ✅ تحليل دقيق لوضعك المالي
- ✅ تتبع الدخل والمصاريف بذكاء
- ✅ إدارة الالتزامات الثابتة والرواتب تلقائياً
- ✅ تجربة استخدام احترافية وسلسة
- ✅ دعم كامل للوضع الداكن (Dark Mode)
- ✅ تثبيت على الهاتف كتطبيق عادي (PWA)

---

## 🎨 الهوية البصرية

### الألوان الأساسية
- **Primary**: `#ff2d78` - الوردي الزاهي (الإجراءات والتأكيد)
- **Secondary**: `#00ffcc` - أخضر فيروزي (التوازن والإيجابية)
- **Tertiary**: `#ffe04a` - أصفر دافئ (التنبيهات والملاحظات)
- **Error**: `#ff4444` - أحمر الأخطاء والمصاريف
- **Success**: `#4caf50` - أخضر الإيرادات والإيجابيات
- **Background**: `#0a0a12` - أسود عميق (خلفية الوضع الداكن)

### الخطوط
- **العربية**: Almarai (وضوح عالي ومظهر عصري)
- **الأرقام/الإنجليزية**: JetBrains Mono (احترافية مالية)

## 🚀 الميزات الأساسية

### 1. التحقق من الهوية والمستخدمين
- ✅ تسجيل دخول بسيط (اسم وبريد)
- ✅ حفظ البيانات محلياً على الجهاز
- ✅ نسخة تجريبية فورية

### 2. لوحة القيادة الرئيسية (Dashboard)
- 📊 **صافي الثروة** - مجموع أرصدة الحسابات
- 💹 **تحليل الدخل والمصاريف** - نظرة سريعة
- 🏦 **الحسابات** - عرض جميع محافظك
- 📈 **آخر العمليات** - أحدث 5 تحويلات
- 📉 **ملخص الميزانية** - توزيع المصاريف حسب الفئات

### 3. إدارة الحسابات (Accounts)
```
أنواع الحسابات المدعومة:
├── 💵 نقد (Cash)
├── 🏦 حساب بنكي (Bank Account)
├── 💳 بطاقة ائتمان (Credit Card)
└── 🏪 حساب توفير (Savings Account)
```

### 4. سجل العمليات (Transactions)
- ➕ إضافة مصروف أو دخل
- 📝 تعديل العمليات السابقة
- 🔍 البحث والتصفية
- 📅 عرض حسب التاريخ
- 💬 إضافة ملاحظات

### 5. الفئات المرنة (Categories)
```
فئات المصاريف الافتراضية:
├── 🛒 مقاضي
├── ⚡ فواتير
├── 🍔 مطاعم
├── 🚗 مواصلات
├── 🎬 ترفيه
└── ⚕️ صحة

فئات الإيرادات:
├── 💼 راتب
└── 📈 استثمارات
```

### 6. الالتزامات الثابتة الذكية ⭐
**هذه أهم ميزة في التطبيق:**

عند إضافة مصروف (مثل قسط سيارة، جمعية، اشتراك):
1. ✅ تفعيل خيار "التزام ثابت"
2. ✅ تحديد تاريخ الانتهاء
3. ✅ اختيار نمط التكرار (يومي/أسبوعي/شهري/سنوي)
4. ✅ النظام يخصم المبلغ تلقائياً في التاريخ المحدد

### 7. الراتب التلقائي
- 📅 تحديد قيمة الراتب الشهري
- 📍 تحديد يوم الدفع
- 💰 إضافة تلقائية كدخل في الحساب

### 8. تطبيق الويب التقدمي (PWA)
- 📱 تثبيت على الهاتف كتطبيق عادي
- 🔄 تحديثات تلقائية
- 📴 العمل بدون إنترنت (قريباً)
- 🎨 أيقونة مخصصة على الشاشة الرئيسية

## 🛠️ كيفية الاستخدام

### التثبيت والتشغيل
1. **فتح التطبيق**
   ```
   افتح الملف index.html في المتصفح
   ```

2. **التسجيل**
   - أدخل اسمك والبريد الإلكتروني
   - أو اختر "جرب النسخة التجريبية" للبيانات التجريبية

3. **التثبيت على الهاتف**
   - انقر على الخيارات (⋮) في متصفحك
   - اختر "تثبيت التطبيق"
   - سيظهر أيقونة على شاشتك الرئيسية

### إضافة عملية مالية
```
1. انقر على زر ➕ (FAB)
2. اختر نوع العملية (مصروف/إيراد)
3. اختر الفئة
4. أدخل المبلغ
5. اختر الحساب
6. أضف ملاحظة (اختياري)
7. إذا كانت التزام ثابت، فعّل الخيار
8. انقر "إضافة العملية"
```

### إنشاء التزام ثابت
```
1. أثناء إضافة عملية، فعّل "التزام ثابت"
2. حدد تاريخ الانتهاء
3. اختر نمط التكرار
4. سيتم الخصم تلقائياً
```
